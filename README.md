# تطبيق تأمين السفر - Travel Insurance App

تطبيق ويب مستقل لإصدار وثائق تأمين سفر إلكترونياً.

## المميزات

- ✅ حساب المستخدم (تسجيل دخول/تسجيل جديد)
- ✅ إدارة الملف الشخصي
- ✅ إنشاء وثائق تأمين سفر
- ✅ احتساب السعر تلقائياً حسب المدة والوجهة
- ✅ إصدار الوثائق الإلكترونية
- ✅ إدارة الوثائق الصادرة
- ✅ تحميل شهادات التأمين بصيغة PDF
- ✅ واجهة مستخدم بسيطة وسهلة (موبايل أولاً)
- ✅ دعم اللغة العربية (RTL)

## التقنيات المستخدمة

- **React 19** - مكتبة UI
- **React Router** - للتوجيه
- **Tailwind CSS** - للتصميم
- **Shadcn UI** - مكونات UI
- **GSAP** - للحركات (إن لزم الأمر)
- **jsPDF** - لتوليد ملفات PDF
- **Vite** - أداة البناء

## البدء

### التثبيت

```bash
npm install
```

### التشغيل

```bash
npm run dev
```

سيعمل التطبيق على `http://localhost:5173`

### البناء للإنتاج

```bash
npm run build
```

## الألوان المخصصة

التطبيق يستخدم الألوان الخاصة بالشركة:

- `--btn-primary`: اللون الأساسي للزر الرئيسي
- `--btn-secondary`: اللون الثانوي للزر
- `--btn-text`: لون نص الأزرار

يمكن تعديل هذه الألوان في ملف `src/index.css`

## الاستخدام

1. **تسجيل الدخول**: أنشئ حساب جديد أو سجل الدخول
2. **إنشاء وثيقة**: املأ بيانات المسافر والرحلة
3. **مراجعة السعر**: يتم احتساب السعر تلقائياً
4. **إصدار الوثيقة**: اضغط على إصدار الوثيقة
5. **إدارة الوثائق**: عرض وتحميل جميع الوثائق الصادرة

## ملاحظات

- البيانات حالياً تُحفظ في localStorage (للاستخدام المحلي فقط)
- في الإنتاج، يجب ربط التطبيق بخادم API حقيقي
- توليد PDF يدعم العربية ولكن قد يحتاج خطوط عربية محسنة
